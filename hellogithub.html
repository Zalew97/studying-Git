С помощью Git программы и разрабочики оринетируются в коде и отслеживают изменения. Git помогает вернуть файлы в исходное состояние и видетль изменения, внесенные в определенный период Git - это системы контроля версий, а GitHub - онлайн-сервис, по сути социальная сеть. Одна из основных целей GitHub - быть единым местом для проектов с исходным кодом. Предполагается,что пользователь делится чем-то полезным, а другие люди смогут участвовать в разработке.
